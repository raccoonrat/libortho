Add experiment data collection and visualization

Add comprehensive data collection and visualization system for all three
core experiments. Generates publication-ready figures (PNG, 300 DPI) and
detailed numerical results (JSON, CSV).

What it does:
- Collects detailed metrics from all three experiments
- Generates 4-panel figures per experiment (training, results, ratios, summary)
- Exports JSON (complete data) and CSV (summary tables)
- Validates success/failure criteria automatically

Files added:
- experiments/run_experiments_with_visualization.py: Main script
- experiments/generate_paper_figures.py: Quick generator
- experiments/EXPERIMENT_VISUALIZATION_README.md: Usage guide
- docs/EXPERIMENT_RESULTS_TEMPLATE.md: Paper template
- docs/EXPERIMENT_DATA_COMPLETE.md: Completion report

Dependencies:
- matplotlib, seaborn (added to requirements.txt)

Fixes:
- NameError in save_results() (results -> self.results)
- Unicode font warnings (PASS/FAIL instead of checkmarks)

Tested and working. All figures are paper-ready.
