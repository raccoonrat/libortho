Add experiment visualization and data collection

Add script to run all three experiments with automatic data collection
and publication-ready figure generation.

- Collects detailed metrics (errors, ratios, sparsity)
- Generates 4-panel figures per experiment (PNG, 300 DPI)
- Exports JSON (complete data) and CSV (summary tables)
- Validates success criteria automatically

Files:
- run_experiments_with_visualization.py: Main script
- generate_paper_figures.py: Quick generator
- Documentation and templates

Dependencies: matplotlib, seaborn

Fixes: NameError and Unicode font warnings

Tested. Figures are paper-ready.
